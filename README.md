# Card Game Project 🃏
--------------------------

Welcome to the **Card Game Project**! 🎮 This is an exciting card battle game still under development, inspired by classic card games like **Yu-Gi-Oh!**. It features **boss battles**, **unique cards**, and **strategic gameplay** with exciting mechanics!

![image](https://github.com/user-attachments/assets/6cd85825-8f9d-4133-a838-b67a55b86db2)

## 🔥 Solo Project 🔥
This project was developed independently as part of my personal and freelancing experience.

## 🎮 Gameplay Details 🎮

- **Operating System**: Mobile 📱
- **Controls**: Touch 📱
- **Game Type**: 2D Game 🌍

## 🌟 Key Features 🌟

- **Card Battle Mechanics** 🃏: Engage in strategic card-based gameplay where you summon powerful cards and defeat enemies with deck-building tactics.
- **Boss Fights** 👑: Battle against **multiple bosses**, each with unique abilities, decks, and strategies to challenge your skills.
- **Yu-Gi-Oh!-inspired Card View** 🧑‍🎤: A visually inspired card interface and battle system reminiscent of **Yu-Gi-Oh!** for a tactical and fun experience.
- **Multiple Card Types** 🔮: Collect and customize your deck with various cards—ranging from **attack** 💥, **defense** 🛡️, to **special abilities**.
- **Deck Building** ⚒️: Create unique decks tailored to different playstyles, whether offensive, defensive, or hybrid.
- **Under Development** ⚙️: The game is actively being improved with new features and mechanics added in future updates.

## 🔧 Technologies Used 🔧

- **Unity 2D** 🎮
- **C# Programming Language** 💻
- **Custom Card Engine** 🃏
- **UI/UX Design** 🎨

## 🚧 Current Development Status 🚧

- **Core Gameplay**: The fundamental card battle mechanics and deck-building system are in place.
- **Boss Battles**: Several bosses have been implemented, each with distinct mechanics. More will be introduced in later updates.
- **Card System**: The card visuals, effects, and interactions are still being refined—expect new card types and improved interactions soon.
- **User Interface (UI)**: The UI is undergoing adjustments to provide a seamless gameplay experience.

## 📚 Story of this game 🏚️
--------------------------

This project holds a special place in my **freelancing** journey! 💖 I had the pleasure of working with one of my **best clients**, who shared a deep love for classic card games like **Yu-Gi-Oh!** 🎮🀄️. Our vision was to recreate the thrill of **strategic card battles** while adding innovative mechanics to enhance the experience. 🔥

However, my time with this project came to an unexpected halt 🎪. Just as I was nearing the final development stages ⌛, I received an offer for a **new opportunity** 🌐✨—a top-secret venture that I couldn’t turn down 🎥🚫 (no spoilers!). Because of this, I had to step away, leaving the project incomplete ❌. While it was bittersweet, the excitement of what came next made it a thrilling transition.

Although I couldn't finish this game, the collaboration was **unforgettable**—a journey filled with creativity, growth, and teamwork 💞. This project remains a key milestone in my career, carrying valuable lessons and incredible memories. 🕊️
